    Nama saya adalah Ide Kristiani Zega, akrab dipanggil Kristin. Saya berusia 20 tahun, dan merupakan mahasiswa aktif dari Universitas Prima Indonesia.

    Saya sama sekali tidak ada ketertarikan dengan dunia programming, namun semakin mengenal dunia programming membuat saya paham bahwa dunia programming justru menarik saya untuk mempelajarinya lebih dalam. Saya memang masih belum mengerti apa-apa, terkadang melihat orang-orang yang begitu cepat menguasai materi programming hanya dalam beberapa bulan atau sudah begitu berpengalaman hanya dalam kurun waktu 2 tahun membuat saya kagum. Iri? Insecure? Menyerah? Tentu tidak. Saya merasa bahwa ada waktu tepat ketika saya merasa saya sudah paham dan berpengalaman dalam dunia programming. Waktu orang lain mungkin terlalu cepat, tapi bukan berarti waktu saya terlalu lambat. Saya percaya pada proses dan hasil.

    Bosen belajar dan menyerah itu wajar, tapi tetap mencoba dan berusaha adalah tanda bahwa saya tidak akan pernah gagal.
